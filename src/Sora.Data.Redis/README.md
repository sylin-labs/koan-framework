# Sylin.Sora.Data.Redis

Redis provider for Sora data: key-value storage with simple scanning, options binding, and health checks.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Redis
```
