# Sylin.Sora.Data.Postgres

PostgreSQL provider for Sora relational data: Dapper-based SQL integration with guardrails and governance.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Postgres
```
