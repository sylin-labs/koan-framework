# Sylin.Sora.Messaging.Abstractions

Core messaging interfaces and primitives for Sora (messages, envelopes, inbox/outbox contracts).

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Messaging.Abstractions
```
