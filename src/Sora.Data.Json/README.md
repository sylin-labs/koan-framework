# Sylin.Sora.Data.Json

JSON file-backed data provider for Sora: simple repository for demos, tests, and local development.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Json
```
