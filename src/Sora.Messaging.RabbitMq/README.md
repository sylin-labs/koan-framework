# Sylin.Sora.Messaging.RabbitMq

RabbitMQ transport for Sora Messaging: connection factory, publisher, and consumer helpers.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Messaging.RabbitMq
```
