# Sylin.Sora.Data.Sqlite

SQLite provider for Sora relational data: lightweight local development and testing support.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Sqlite
```
