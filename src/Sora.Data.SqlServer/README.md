# Sylin.Sora.Data.SqlServer

SQL Server provider for Sora relational data with JSON-projection pushdowns, guardrails, and governance.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.SqlServer
```
