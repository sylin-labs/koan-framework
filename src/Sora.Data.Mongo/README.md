# Sylin.Sora.Data.Mongo

MongoDB data provider for Sora: options binding and repository integration for document databases.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Mongo
```
