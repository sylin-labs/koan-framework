# Sylin.Sora.Data.Weaviate

Weaviate vector database provider for Sora: schema utilities, filter translation, and health.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Weaviate
```
