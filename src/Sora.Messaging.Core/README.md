# Sylin.Sora.Messaging.Core

Core messaging abstractions and helpers for Sora: bus configuration, auto-registration, and options binding.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Messaging.Core
```
