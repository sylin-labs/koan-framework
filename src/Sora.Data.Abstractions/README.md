# Sylin.Sora.Data.Abstractions

Data abstractions for Sora: repository contracts, entities, and filtering interfaces.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Data.Abstractions
```
