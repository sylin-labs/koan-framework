# Sylin.Sora.Web.Swagger

Swagger/OpenAPI integration for Sora Web: idempotent registration, config-driven enablement, and friendly defaults.

- Target framework: net9.0
- License: Apache-2.0

## Install

```powershell
dotnet add package Sylin.Sora.Web.Swagger
```
